# quiethelp-front

descripción

Frontend del proyecto QuietHelp desarrollado con Flutter.
Permite a los alumnos enviar mensajes de forma anónima y a los orientadores consultar la información desde la interfaz.


- - -
tecnologías

- Figma
- Flutter
- Dart

- - -
planificación (walkthrough – febrero)

semana 1:
- diseño completo de la interfaz en figma
- creación del proyecto Flutter


semana 2:
- estructura básica de pantallas en Flutter
- configuración inicial

semana 3:
- pantallas de envío de mensajes
- conexión con el backend

semana 4:
- envío y recepción de datos
- pruebas de la aplicación
